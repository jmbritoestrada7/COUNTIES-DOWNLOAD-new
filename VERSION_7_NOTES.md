# Version 7 – STR by Acreage Range

- Replaces the incorrect minimum/maximum STR controls with a **Color counties by** selector.
- Supported metrics: Average STR, STR 2–5, STR 5–10, STR 10–20, STR 20–60, STR 60–100, and STR 100+ acres.
- County colors, badges, statistics, highest-county ranking, popup highlight, and legend update automatically when the selected acreage range changes.
- Counties without data for the selected acreage range remain active and display in neutral gray.
- The selected range is autosaved to R2 and synchronized between users opening the same project.
- Existing projects default to Average STR until a different acreage range is selected.
- Retains the Version 6 mobile interface and English file picker.
